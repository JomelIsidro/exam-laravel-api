

## Laravel 11

PHP version: 8.2

## Installation

git clone https://github.com/JomelIsidro/exam-laravel-api.git

cd <backend-directory>

composer install

cp .env.example .env

php artisan key:generate

MySQL database
- create database "exam" on your local 

php artisan db:seed --class=UserSeeder

php artisan migrate

php artisan serve
